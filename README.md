# Stock.Tech
Desenvolvimento da Sprint 1 do Trabalho Interdisciplinar do curso de Sistemas de Informação (PUC Minas) focado na criação do dashboard corporativo Stock.Tech. A entrega engloba a estruturação da interface estática com HTML/CSS e a programação da interatividade utilizando JavaScript. O destaque técnico desta etapa é a implementação de um sistema CRUD completo para notas operacionais com persistência de dados local (Web Storage API / LocalStorage), além da exibição dinâmica de indicadores e gráficos interativos para desktop, estabelecendo uma base sólida para as próximas integrações.
